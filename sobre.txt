Linguagens - Javascript, python 
Tecnologias :
        Editor de código: Visual Studio Code
        Front-end: HTML5, CSS, Javascript, BootStrap
        Design: Canvas, Figma
        Repositorio & Versionamento: Github, Git
        Dominio DNS & Hospedagem: Netlify.app
        Imagens: Unsplash, Pinterest
        Icons: icons8.com
        Geradores Ficticios: 4devs.com
        Automação de processos: Robot, Python e SeleniumLibrary
        Segurança & Prevenção: Exiftools, whois, ViewDNS

Segurança do site:
        Minimizar vulnerabilidades de SQL injection
        Evitar deixar dados no codigo fonte; exemplo (email para contato, telefone, dados pessoais como nome completo)
        Evitar deixar emails no site que tenha dados sensiveis
        Evitar fazer o upload fotos que possam ter metadados (como diretório, geolocalização e etc)
        Verificar e fortalecer a segurança do site, capturando bugs e corrigindo

Sobre o projeto:

           Esse é um projeto criado por mim, para fins academicos.
           ele tem como intuito servir como Site para cadastro e captura de dados
           será usado para fins academicos junto a engenharia social

Sobre as Tecnologias:

        o Front-end será feito com html, css, Javascript e BootStrap, que sao ferramentas para construção de sites para web

        o Design será feito no figma, que é uma ferramenta para edição e construção de layout (https://www.Figma.com)

        o Repositorio para hospedar o codigo-fonte sera o github, e o Versionamento para deploy sera o github
                para acessar o github: (https://github.com)
                para fazer o download do git na maquina, entre no link e siga as instruçoes do site oficial: (https://git-scm.com/downloads)

        o Dominio do site sera feito no Netlify, um sistema de Hospedagem gratuita e com subdominio editavel

        as Imagens do projeto serão baixadas do Unsplash e do Pinterest, sites usados para download de imagens gratuitas, sem copyright e de livre uso

        Caso seja necessario, sera usado o 4devs para gerar dados ficticios 

        A Prevenção sera feita utiizando ferramentasde pesquisas, como o exiftools, whois e ViewDNS:

                exiftools: uma ferramenta capaz de ler metadados de uma Imagem (https://exif.tools)
                Whois: uma ferramenta capaz de consultar um Ip, Dominio ou nomes de um site, retornando informações sobre o mesmo (https://who.is)
                ViewDNS: uma ferramenta capaz de consultar um IP de Dominio do site e retornar informações sensiveis (https://viewdns.info)
